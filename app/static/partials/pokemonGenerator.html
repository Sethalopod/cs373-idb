How it was done

By loading in all the flavour text from the Pokemans API, we were able to generate new ones with Markov Chains using the markovify library. With these new flavour texts for moves and pokedex entries, you are prompted to create a new pokemon! We are planning on transferring the information to a pokemon trading card template and implementing sharing capabilities.
